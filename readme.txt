The program is executed in python. To execute the program, please ensure that the
"numpy", "scikit-learn" and "nltk" libraries are installed, and the programs are
executed in the same directory that it has been unzipped from.

Before running the program, please open a python shell and run the following
commands :

    import nltk
    nltk.download()

Download all the packages when the pop-up is displayed.

Running The Program

1. In the command line, navigate to the working directory.
2. Open python with the file name "approach_*.py", where * will be the approach
number.
